# Resolution-Game_Dev-Week-4

This game was made using Godot for [Resolution Game Dev](https://resolution.hackclub.com/app/pathway/game_dev), a [Hack Club](https://hackclub.com/) YSWS.
It’s fully open source, and you’re free to use it however you like.

This is a simple top-down game with not many features, but it’s pretty cool.
It has 8-directional movement, and the animation changes depending on which direction you're moving in also When you start the game coins spawn that you can collect, but they don’t do anything.
There is also a house when you get close, its door opens and you can enter it, which sends you to another scene thes the house from inside
